# Hướng dẫn cài đặt dự án

 - Clone về máy 
    `git clone https://github.com/Tuannv18112003/du_an_thuc_tap.git`

- Tiếp tục chạy các lệnh sau
    `cp .env.example .env`

    `composer install`

    `npm install`

    `php artisan migrate`

    `php artisan db:seed`

    `php artisan vietnamzone:import`

    `php artisan serve`
